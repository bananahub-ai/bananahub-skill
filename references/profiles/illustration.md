# Profile: 插画与漫画 (Illustration)

## 触发关键词

插画、漫画、动漫、卡通、手绘、二次元、Q版、赛博朋克风、像素画、扁平化、矢量、水彩画、油画、素描、涂鸦、anime、manga、cartoon、pixel art、comic

## 补全维度

基础优化完成后，检查并按需补全以下维度（仅补全用户未提及的）：

### 画风指定
- 用户提到"动漫/二次元" → anime style / manga style
- 用户提到"卡通" → cartoon illustration
- 用户提到"手绘" → hand-drawn illustration
- 用户提到"水彩" → watercolor painting
- **用户无明确风格 → 不推断具体画风**，而是在增强确认时提供选项（如 anime / digital illustration / watercolor / cartoon），让用户选择
- 不要在用户已指定风格时覆盖

### 线条与渲染风格
- 动漫 → clean lines, cel-shading
- 水彩 → soft edges, visible brushstrokes, wet-on-wet blending
- 素描 → pencil strokes, hatching, grayscale
- 像素画 → visible pixels, limited color palette, retro aesthetic

### 配色方案
- 未指定配色 → 根据画风推断合理默认
- 动漫 → vibrant colors（除非氛围偏暗）
- 水彩 → soft pastel tones
- 赛博朋克 → neon colors, dark background
- 不主动限制配色，仅在画风有强烈配色倾向时补充

### 背景处理
- 角色设计/立绘 → 建议 simple clean background / white background
- 场景插画 → 根据描述补充环境细节
- 漫画格 → describe panel layout if relevant

## 示例优化

**用户输入**: 一个拿着魔法杖的小女巫

**基础优化结果**: A little witch holding a magic wand

**增强后结果（用户未指定风格时，提供选项）**:
> 检测到插画类意图，但未指定画风。建议选择：
> 1. anime style — 日系动漫风
> 2. digital illustration — 数字插画
> 3. watercolor painting — 水彩风格
> 4. cartoon illustration — 欧美卡通

**用户选择 anime 后**: A little witch holding a magic wand, anime style, vibrant colors, clean lines with cel-shading, simple gradient background, cheerful and whimsical mood

## 注意事项

- 插画风格种类极多，不确定时宁可不加风格词，让用户选择
- 不要把写实摄影的镜头语言（85mm lens, bokeh）用在插画上
- 漫画/动漫中的文字通常是画面元素，注意保留原文不翻译
